DarkPython

There's a problem with programming education. Most schools use old technology which teaches terrible practices. DarkPython is a fully featured IDE designed for kids programming in Python + Pygame. It is programmed in wxWidgets, Scintilla, and Pygame. Some of it's many features:

  • A tabbed editor.
  • Syntax Highlighting.
  • A full featured debugger. Features which are currently being worked on:
  • A pygame preview window.
  • A GUI debugger with line highlighting.
